# React.js Tailwind Boilerplate with Vite

A production-ready boilerplate for building React applications with Tailwind CSS and Vite. This setup provides a fast, modern development environment with minimal configuration, allowing you to start building immediately.

## Features
- **Vite** - Next generation frontend tooling with instant server start and fast hot module replacement (HMR)

- **React 19** - Latest version with functional components and hooks

- **Tailwind CSS 4** - Utility-first CSS framework with JIT compiler for optimal performance

- **PostCSS & Autoprefixer** - CSS processing and vendor prefixing

- **ESLint** - Code linting with recommended React and Vite rules

- **Prettier (optional)** - Code formatting (can be added as needed)

- **Production optimized** - Build outputs are minified and optimized for deployment

## Prerequisites
- Bun
- bun package manager

## Getting Started
Clone the repository and install dependencies:
```bash
git clone https://github.com/Pranto-Paul/react-js-tailwind-boilerplatecode
cd react-js-tailwind
bun install
```

Start the development server:
```bash
bun run dev
```
Your application will be available at *http://localhost:5173*
